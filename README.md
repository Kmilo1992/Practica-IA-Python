# 🤖 AI LED Control Project

## 🎯 Objective
Control 5 LEDs using Artificial Intelligence and hand detection.

---

## 📁 Required Files

Make sure you have the following files in the same folder:

- `app.py`
- `program.ino`
- `hand_landmarker.task`

---

## 💻 Requirements

Install Python (version 3.10 or higher recommended).

---

## 📦 Install Libraries

Open a terminal and run:

```bash
pip install pyserial
pip install mediapipe
pip install opencv-python
